To clone repo:
1. copy clone link of repo from github
2. use terminal to navigate to the folder you want to clone into
3. clone repo from github with the command: git clone https://github.com/furbus-the-cactus/data-quest.git


Work flow using git:
1. make changes to your code
2. add the changes you made to git with command: git add <files>  # or use git add . to add all files
3. commit the changes with command: git commit -m "<commit message here>"
4. push to github with command: git push origin master
